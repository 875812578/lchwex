# C预处理器如何工作

C预处理器是一个模板处理系统，它主要用于让C代码编程更加容易。
实际使用它来作为代码生成器来创建模板化的代码片段

工作原理：

如果你给它一个文件，比如`.c`文件，它会处理以`#`字符开头的各种文本。
当它遇到定义好的文本时，就会对输入文件中的文本做对应的替换。

可以对ex17中的C文件使用下面命令来查看预处理器所做的事情：

`cpp ex17.c | less`

从输出的信息中，你能看到预定义的MAX_DATA都已经被替换了。
